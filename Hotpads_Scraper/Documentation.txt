# Hotpads Scraper Documentation

## Prerequisites
1.  **Python 3.10+**: Ensure you have a modern version of Python installed.
    *   Download: https://www.python.org/downloads/
2.  **Zyte API Key**: You need a valid Zyte API key.
3.  **Supabase Account** (optional): For database storage.

## Installation

1.  **Open Terminal**: Navigate to the project folder.
    ```bash
    cd path/to/Hotpads_Scraper
    ```

2.  **Create Virtual Environment** (Recommended):
    ```bash
    python -m venv venv
    ```

3.  **Activate Virtual Environment**:
    *   **Windows**:
        ```bash
        venv\Scripts\activate
        ```
    *   **Mac/Linux**:
        ```bash
        source venv/bin/activate
        ```

4.  **Install Dependencies**:
    ```bash
    pip install -r requirements.txt
    ```

## Configuration

1.  **API Key**:
    *   The project uses a `.env` file to store your API key securely.
    *   Ensure the `.env` file exists in the root directory and contains:
        ```
        ZYTE_API_KEY=your_actual_api_key_here
        SUPABASE_URL=your_supabase_url_here
        SUPABASE_SERVICE_KEY=your_supabase_service_key_here
        ```

2.  **Input Locations**:
    *   Edit `input/locations.csv` to add or remove locations.
    *   Format: `location` (header), followed by rows like `chicago il` or `miami fl`.

## Running the Scraper

### Windows
*   Double-click `runner.bat`.

### Command Line (Windows/Mac/Linux)
Make sure your virtual environment is activated, then run:
```bash
scrapy crawl hotpads_scraper
```

## Output
*   CSV file saved to: `output/Hotpads_Data.csv`
*   Database records: `hotpads_listings` table in Supabase (if configured)
